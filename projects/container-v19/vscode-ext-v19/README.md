# vscode extension example !

Extension example

INSTALLATION NOTES
------------------

  1. To install extension in your vscode:
- install  dependencies: 
```
npm i -g vsce typescript
```
- use command: 
```
npm install && tsc && npm run build:install
```
  2. Reload your vscode. Done.
